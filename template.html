Attributes

The library has a number of scripts for cheekily extending the default behaviour of DOM elements, accessed by adding 'behaviour' attributes.

This library provides no style to make these behaviours do anything visual; the behaviours largely add and remove classes: the rest is up to your CSS. This is deliberate. By seperating style from behaviour many different styles of interactive – a menu, a dialog, a tooltip – can share popable behaviour, for example.
